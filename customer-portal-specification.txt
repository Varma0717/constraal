Constraal Customer Portal Specification (Constraal Account)
Portal URL
constraal.com/account
constraal.com/login

Purpose:
 Allows customers to manage their account, billing, purchases, services, support, security, and platform interaction.

1. Account Dashboard
URL
/account

Displays
Welcome message


Account status


Active subscriptions


Recent activity


Notifications


Example:
Welcome, Ahmed

Account Status: Active
Subscriptions: 1 Active

Recent Activity:
Login from Phoenix, AZ
Subscription renewed


2. Profile Management
URL
/account/profile

Customers can:
View profile info


Edit name


Edit email


Change password


Add phone number


Upload profile picture


Profile Data
Name
Email
Phone number
Account creation date
Last login


3. Security Management
URL
/account/security

Features
Change password


Enable Two-Factor Authentication (2FA)


View login activity


Log out other sessions


Login Activity Example
Chrome – Phoenix, AZ – Feb 16, 2026
Safari – New York, NY – Feb 14, 2026


4. Billing and Subscription Management
URL
/account/billing

Customers can:
View active subscriptions


View billing history


Cancel subscription


Upgrade or downgrade plan


Subscription Data
Plan name
Status
Price
Next billing date


5. Payment Method Management
URL
/account/billing/payment-methods

Customers can:
Add payment method


Remove payment method


Set default payment method


Example:
Visa ending in 4242
Expires 12/2027


6. Invoice and Payment History
URL
/account/billing/invoices

Customers can:
View invoices


Download invoices


Invoice example:
Invoice ID: INV-00231
Date: Feb 1, 2026
Amount: $29.00
Status: Paid


7. Orders and Purchase History
URL
/account/orders

Shows all customer purchases.
Example:
Constraal Autonomous System
Purchased: Jan 15, 2026
Status: Completed

Customers can:
View order history


View invoice


Track order status



8. Service and Platform Access
URL
/account/services

Displays customer services and platform access.
Example:
Constraal Platform Access: Active
Automation Services: Enabled
Account Tier: Professional


9. Notifications Center
URL
/account/notifications

Shows:
Billing notifications


System updates


Platform announcements


Example:
Your subscription renews tomorrow


10. Support Center
URL
/account/support

Customers can:
Submit support requests


View support tickets


Track ticket status


Example:
Ticket ID: #2314
Subject: Billing question
Status: Open


11. Account Activity History
URL
/account/activity

Shows account activity.
Example:
Logged in
Updated password
Subscription created


12. Privacy and Data Management
URL
/account/privacy

Customers can:
Download account data


Delete account (optional)


Manage privacy settings



13. Email and Notification Preferences
URL
/account/preferences

Customers can enable/disable:
Billing emails


Security alerts


Platform updates


Marketing emails



14. Account Settings
URL
/account/settings

Customers can:
Update account settings


Manage preferences



15. Logout
URL
/logout

Ends session securely.

Orders:
